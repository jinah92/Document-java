# 변수와 상수



### 변수

---

- 변하는 값

- 선언 이후에도 값을 변경할 수 있음

- 변수 명은 소문자로 시작
(두 개의 단어가 있는 경우 마지막 단어의 첫글자는 대문자로 명명)

  
  
- ```java
int count; //변수 선언
  	int a = 10; //선언과 동시에 대입
    	double count2; //double형 변수 
  
    	count = 10;
    	count = 20;
  
    	System.out.println(count); //20 출력 
  ```
  
    	
  

### 상수

---

- 변하지 않는 값

- 한번 선언되면 값을 변경할 수 없음

- 상수 명은 대문자로 명명
(두개의 단어가 있는 경우 사이에 _ )

  
  
- ```java
	final COUNT = 10;
  	COUNT = 20; //error
  ```
