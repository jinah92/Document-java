# 자바의 개요



## 자바의 탄생

---

- 1995년 썬 마이크로시스템즈에서 발표한 객체지향 언어

## 자바 언어 특징

---

- 타 언어에 비해 배우기 쉽다
    - C와 C++언어의 문법을 기본으로 차용하여 개발
    - C와 C++이 가진 포인터와 다중 상속을 제거
    - C와 C++에 비해 쉬운 언어
- 플랫폼에 독립적
    - JVM만 있으면 윈도우, 리눅스, MAC 등 어떤 플랫폼에서든지 실행이 가능
- 객체 지향 프로그래밍
- Garbage Collector로 사용되지 않는 메모리를 자동적으로 정리
